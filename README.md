# skardu-tourism
tourism 

🌍 Skardu Tourism Website

This project is a Tourism Website for Skardu built using HTML, CSS, and JavaScript.
The website shows the beauty of Skardu and nearby tourist places such as Deosai Plains, Satpara Lake, and Ghanche Valley.

📌 Features

Beautiful tourism homepage

Image gallery of Skardu tourist places

Responsive design

Simple and clean UI

Easy navigation

🛠 Technologies Used

HTML5

CSS3

JavaScript
